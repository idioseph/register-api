## Run Locally

Clone the project

```bash
  git clone https://github.com/idioseph/register-api.git
```

Go to the project directory

```bash
  cd register-api/register-api
```

Install dependencies

```bash
  npm install
```

Start the server

```bash
  npm run start
```
